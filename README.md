# 🎭 繁笼RPG系统 · 通用版

> 给你的语C/TRPG群装一套真正的 RPG 管理系统。
> 基于 **LLbot + 青果核** 框架，数据库驱动，适配任何世界观。

---

## ✨ 核心特性

- 🎭 **角色养成** — 档案创建、8大属性、装备穿戴、每日签到训练
- 💰 **经济系统** — 双货币体系、商店购买、背包管理、货币兑换
- 📖 **戏录存档** — 自动/手动录入、发言统计、活跃排行
- 🎲 **TRPG骰子** — 多面骰、属性检定、大成功/大失败判定
- 📇 **名片同步** — 自定义模板、多群自动同步
- ⚙️ **完全可配置** — 所有术语、数值、商品均通过数据库自定义，无需改代码

> 货币叫"灵石"还是"金币"，属性叫"颜值"还是"悟性"，全部自己定。古风、修仙、玄幻，任何世界观都能用。

## 🚀 快速开始

### 环境要求

- [LLbot](https://github.com/OlivOS-Team/OlivOS) + 青果核框架
- 环境搭建教程参考：[临皋闲人老师的前置教程](https://gcnjvcr0wzab.feishu.cn/wiki/M8Svw3iqeixHOLkBnwMcoo4Gnmd)

### 安装步骤

**1. 放置插件文件**

将插件文件夹放入青果核插件目录：

```
OlivOS/plugin/app/
├── fanlong_core/      ← 必须
├── fanlong_rpg/       ← 必须
├── fanlong_drama/     ← 可选
├── fanlong_dice/      ← 可选
└── fanlong_card/      ← 可选
```

**2. 重启插件**

重启青果核，控制台确认所有插件加载成功。

**3. 配置管理员**

首次启动后，在数据库 `admins` 表中添加你的 QQ 号，否则无法使用管理员指令。

**4. 发送重载配置**

用管理员账号在群内发送 `重载配置`，使数据库配置生效。

## 💬 常用指令速查

### 玩家指令

| 指令 | 说明 |
|------|------|
| `创建 [姓名]` | 注册新档案 |
| `档案` | 查看角色详情 |
| `属性` | 查看八维属性 |
| `商店` | 查看商品列表（货币改名后口令同步变更） |
| `购买 [物品]` | 购买商品 |
| `签到` | 每日获取资源 |
| `训练` | 随机提升属性 |
| `r` / `.r [属性]` | 掷骰 / 属性检定 |
| `戏录列表` | 查看存档戏录 |
| `帮助` | 查看完整帮助菜单 |

### 管理员指令

| 指令 | 说明 |
|------|------|
| `重载配置` | 刷新数据库配置（修改后必须执行） |
| `录入 @人 字段 内容` | 修改角色档案 |
| `增加 @人 [属性/物品]*N` | 增加属性或物品 |
| `扣除 @人 [属性/货币]*N` | 扣除属性或货币 |
| `加全员 [属性]*N` | 全员增加属性 |
| `发薪资` | 按档案薪资字段全员发钱 |
| `存戏 日期+标题+人物` | 存入戏录 |

## 🖥️ B端后台管理

附带网页版后台管理系统 `fanlong-admin-lite`，无需操作数据库，浏览器里即可完成：

- 用户管理、属性查看、背包编辑
- 商店物品增删改
- 术语翻译配置（改"虞元"为"灵石"等）
- 游戏参数设置
- 数据备份与恢复
- 操作日志审计

**部署方式**：参考 [`fanlong-admin-lite/部署教程.md`](fanlong-admin-lite/部署教程.md)

> 修改任何数据后，需在群内发送 `重载配置` 才能生效。

## ⚙️ 自定义配置

所有内容均可通过数据库 `fanlong.db` 或B端后台配置：

| 配置项 | 说明 |
|--------|------|
| 术语翻译 | 修改"虞元""颜值"等全部显示文字 |
| 游戏数值 | 属性上限、签到奖励、训练范围等 |
| 商品配置 | 物品名称、价格、属性加成、制作配方 |
| 档案字段 | 自定义角色档案的显示字段 |
| 自定义回复 | 触发关键词后的机器人回复 |

---

## 📋 注意事项

1. **优先备份数据库** `fanlong.db` ——这是所有用户数据的唯一存储，丢失不可恢复
2. 修改数据库后必须发送 `重载配置` 才能生效
3. 大批量修改前建议暂停机器人，避免并发冲突
4. 名片同步功能需在 `fanlong_card/main.py` 中额外配置 API 端口和 Token
